# Tab Harbor プライバシーポリシー / Privacy Policy

最終更新 / Last updated: 2026-07-09

## 日本語

Tab Harbor は、タブと Pin(ブックマーク)を Space 単位で整理するためのブラウザ拡張です。

- **収集するデータ**: ありません。Tab Harbor は利用者のデータを開発者や第三者のサーバーへ
  送信しません。アナリティクス・トラッキングも行いません。
- **保存するデータ**: Pin(タイトルと URL)、Space の設定(名前・色)、Space ごとの
  タブの URL(復元用)。これらはすべてブラウザ内(`chrome.storage.sync` /
  `chrome.storage.local`)にのみ保存されます。`storage.sync` のデータは、
  利用者自身のブラウザアカウントの同期機能によって利用者のデバイス間でのみ同期されます。
- **Arc インポート**: 利用者が明示的に選択したファイルをブラウザ内でローカルに解析します。
  ファイルの内容が外部へ送信されることはありません。
- **権限の用途**: `tabs` / `tabGroups`(サイドバーへのタブ表示と Space 切替)、
  `storage`(上記データの保存)、`sidePanel`(サイドバー UI)、`favicon`
  (サイトアイコンの表示)。
- データの削除は、拡張機能をアンインストールするだけで完了します。

## English

Tab Harbor is a browser extension that organizes tabs and pinned bookmarks into Spaces.

- **Data collection: none.** Tab Harbor does not transmit any user data to the developer
  or any third party. No analytics, no tracking.
- **Data stored**: pins (title and URL), Space settings (name, color), and per-Space tab
  URLs for restoration. All of it stays inside the browser (`chrome.storage.sync` /
  `chrome.storage.local`). Synced data travels only through the user's own browser
  account sync.
- **Arc import** parses a file explicitly chosen by the user, locally, in the browser.
  Nothing is uploaded.
- **Permissions**: `tabs` / `tabGroups` (show tabs in the sidebar, switch Spaces),
  `storage` (persist the data above), `sidePanel` (sidebar UI), `favicon` (site icons).
- Uninstalling the extension deletes all stored data.
